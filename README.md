Velora Project — инди P2P онлайн RP-проект, вдохновлённый SAMP. Разработан на Unity с помощью ChatGPT и других нейросетей.
Для игры впервые необходимо отредактировать персонажа, подключиться к сессии/создать сессию. Готово можно играть!

P2P онлайн игра ( Глобальное подключение ) :
В игре Хост - игрок создавший сессию. Клиент - игрок который подключился к сессии Хоста ( P2P игра ).
Связь с игрой и соединение зависит от Хоста создавшего сессию

Оффлайн / Локальная игра :
В игре предусмотрена игра оффлайн при отсутствии интернета и локальная игра ( игра по одной сети )
Для игры в оффлайн необходимо нажать на кнопку ОФФЛАЙН в главном меню игры. 
Для онлайн игры ( Глобальное подключение P2P ), необходимо нажать на кнопку ПОДКЛЮЧИТЬСЯ, СЛУЧАЙНАЯ ИГРА, СОЗДАТЬ СЕССИЮ.

Возможности игры :
Зарабатывать игровые деньги, общаться с другими игроками, кастомизировать своего персонажа, покупать игровую недвижимость, транспорт.
Можно стать мэром в сессии каждые 15 минут на 15 минут. Можно работать в разных организациях, 
В игре присутствует базовая автомодерация чата с фильтром слов.
В качестве наказания игрока, игрокам будет предложено голосование против выбранного игрока, далее игрок покинет сессию.

В данной игре отсутствуют внутриигровые услуги за реальные деньги, всего добиваются игроки сами.
В качестве добровольной поддержки можно поддержать финансово автора игры за проделанный труд, нажав на кнопку ПОДДЕРЖАТЬ ДОНАТОМ в главном меню,
далее будет переход на платформу с добровольным пожертвованием на DonationAlerts.

Любые модификации в игре произведенные со стороны игрока могут усложнить для него игру или совсем привести к непредвиденным ошибкам.
Рекомендуется не вносить никаких модификаций в игру иначе она может у него не работать.

Автор игры не несет ответственности за чат, ( присутствует базовая автомодерация ), и за действия игроков в игре.

Установить игру на Windows, MacOS, Android можно на этом Github репозитории по ссылке : https://github.com/duplosha/velora_project в разделе releases.
